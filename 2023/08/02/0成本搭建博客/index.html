0成本搭建博客

下载git

除了设置安装路径外，其他可以默认

命令行git -v验证

下载Node.js

同样出来安装路径，其余默认即可

命令行node -v验证

安装hexo

git bash 或 命令行输入 npm install -g hexo-cli

git bash 或 命令行输入 hexo -v

注册github账号

新建仓库

new repositor
Respository name 格式为 账号的名字.github.io

其余可以默认即可。

配置ssh免密登录

删除c:\Users\用户名.ssh文件夹

在c:\Users\用户名 文件夹用git bash 打开,输入 ssh-keygen -t rsa -C github邮件地址, 生成.ssh秘钥，输入后连敲三次回车

生成了一个新的 C:\Users\用户名.ssh文件夹，打开这个文件夹，找到.ssh\id_rsa.pub，打开并复制里面的内容

打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key，把复制的内容粘贴进去，title随便填

git bash 中输入ssh -T git@github.com 不用改

如果提示Are you sure you want to continue connecting (yes/no)?，输入yes，然后会看到：
Hi XXXX! You’ve successfully authenticated, but GitHub does not provide shell access.
看到这个信息说明SSH已配置成功

git config –global user.name 你的github账户名
git config –global user.email 填写你的github注册邮箱

使用hexo

在电脑随便找个地方，git bash 打开 输入 hexo init
会自动生成一些文件。

输入hexo g 生成，hexo就会在public文件夹生成相关html文件
输入hexo s 启动服务，开启本地预览服务

打开浏览器访问 http://localhost:4000 即可看到内容

按住ctrl, 鼠标点击git bash中的链接，即可浏览器打开

将博客部署到 github 个人主页

在博客的根目录下 git bash打开

输入npm install hexo-deployer-git --save

在根目录下的_config.yml 中末尾 改为

在博客根目录使用git bash打开，输入hexo d

https://用户名.github.io/ 即可打开

如果没有问题的话，需要等一会。

自学计算机到极致

发表于2023-08-09|更新于2023-08-14|自学计算机到极致
|阅读量:

存储器

  • 内存
    • 电子产品设计的一个准则是防呆
    • 内存条，内存芯片

内存芯片

外形：一个长方块，上方有个缺口（凹陷），两边有引脚
缺口边两引脚距离：15.24mm(公制单位), 0.6inch(英尺), 600mil(密尔)
缺口邻边两引脚距离：2.54mm(公制单位), 0.1inch(英尺),100mil(密尔)

封装(机械参数，Mechanical Parameter)：外形与尺寸的合成

DATA SHEET（说明书，数据纸）：芯片的所有的参数（机械，电气参数…）都可以在这里查到，每个芯片都会有，一般为电子的，pdf。

引脚向外撇，呈八字的原因：电子产品在工厂进行装配时，都是自动化的，机器手抓两边时正好把引脚捏直了。

规律：引脚的标号， 缺口左边的引脚为1， 左下引脚为14, 右下引脚为15， 右上引脚为28。
从缺口左面开始逆时针转一圈到右边（按字母U的书写顺序）

芯片表面的字：
以视频为例：HM6264ALP-15
HM6264ALP-15为芯片的型号，HM是芯片的生产厂商的代号，6264是芯片真正的型号（相同型号可以换着使用），A是版本信息，L可能是低耗电，P未知，15芯片的性能（存取过程的时间）

芯片：黑块为树脂（耐热塑料）， 内部有一个硅片，电路肉眼看不见，肉眼可以看见一些触点，这些触点通过金属导线(黄金只有一点，引脚会延伸到硅片附近)连接到引脚，引脚是铜的

28引脚接电源正极，14接电源负极，一般是右上接正极，左下接负极。具体看说明书

电气参数（Electrical Parameter）：

  • 静态 DC(直流)
  • 动态 AC(交流)
  • 极限

DC operating conditions 直流参数
supply voltage 电源电压 min typ max

引脚1：NC(No Connection) 无连接引脚， 硅片与引脚没有连接, 有时可以与引脚28连接作为正极，电路板实现。

I/O 1 ~ 8：输入输出(input output), 数字通过I/O进行传输, 接到高电平为1，低电平为0

芯片内部有I 和 O 通过开关来连接I/O 通过信号 i 通过 上划线we 位于引脚27 write enalbe(允许，有效) o 通过 上划线oe 上划线表示需要接到低电平才是enable, 高电平则是disable

电源切断数据会消失，通电不会恢复数据

HM6264可以存储8192(8K) * 8 = 65534(64K, 6264的64就是这个意思)

A0 在 I/O1上面 A12 在 NC下面 A8, A9, A10 在右面

A(Address, 地址)是单向的

引脚26 是 CS2 I/O8 上面 上划线CS1 引脚

(Chip Select)片选，芯片使用时需要被选择 （睡眠状态）消耗的电量非常少

Volatile(易失) Non-Volatile(非易失)

static 放到存储器中一般是不会改变的

Random Access (随机存取)

Memory

SRAM

铁电存储器是非易失的，却是RAM， 断电之后数据是不会消失的，容量却很小

内存条是动态存储器， 高电平只能保持一会儿，一直保持需要在写一遍，“刷新”，硬件会做刷新的功能

波形图（时序图）：加入信号，地址，数据的时间顺序

真值表

电源负端就是”地”, 就是0V

电路板

一块绝缘材料做的薄板

芯片出来的导线，沿着电路板表面走的

拓展知识

电路中电源负极为0电压，电压的参考点 Grpimd(地面，平地)接地点 电压零点。电路中在接入一个电源，正极接在0点，负极的电压就为负数

集成电路的尺寸标注的最小单位为0.001inch(英尺) = 1mil(密尔)

数据从芯片里传出来，叫做”读”出

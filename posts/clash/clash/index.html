下载

archlinux 使用 sudo pacman -S clash

配置

终端输入 clash， 会在 ~/.config/ 下生成一个 clash/ 目录,

由 https://github.com/Dreamacro/maxmind-geoip/releases/latest/download/Country.mmdb, 或 https://go.runba.cyou/ssr-download/Country.mmdb 下载 Country.mmdb

并替换 clash/ 目录下的 Country.mmdb

下载 clash 节点， 替换 clash/目录下的 config.yaml, 下载地址: https://free.datiya.com/

在 .bashrc 添加

1
2

export http_proxy="http://127.0.0.1:7890"
export https_proxy="http://127.0.0.1:7890"

clash web 管理

下载代码

clash-dashboard

1
2
3

mv ~/Download/clash-dashboard.tar.gz ~/.config/clash/
cd ~/.config/clash/
tar zxvf clash-dashboard.tar.gz

修改配置

vim ~/.config/clash/comfig.yaml

添加下面的代码, 可能external-contorller可能已经存在, 直接添加 127.0.0.1 即可

1
2
3

external-controller: 127.0.0.1:9090 # 如果你不是从本机访问，需要从其它机器访问这个Clash Dashboard ,则改为：0.0.0.0:9090
external-ui: ./clash-dashboard # clash-dashboard的路径；
secret: '123' # PaaRwW3B1Kj9 是登录web管理界面的密码，请自行设置你自己的,不要照抄教程中的密码；

重启 clash

网页输入 127.0.0.1:9090/ui 即可打开, 在这里修改的都是临时修改

错误提示

如果没有用， 重启电脑试一试。

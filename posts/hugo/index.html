使用 hugo

hugo new content posts/.../name.md

创建一篇文章, posts可以自定义, 前面不需要斜杠, …可以是多个目录

.Summary

Let Hugo create the summary based on the first 70 words. You can change the number of words by setting the summaryLength in your site configuration.

Manually split the content with a <!–more–> tag in Markdown. Everything before the tag is included in the summary.

Create a summary field in front matter.

大概意思就是设置summary的长度, 使用<!–more–>分割summary的内容和文本内容, 在 front matter (.md开头的内容）创造一个summary

函数指针类型

  1. 函数指针变量

int (*p)(int a, int b)

定义了一个函数指针变量 p, 它可以指向返回值为 int, 形参为 int, int 的函数

int (*p[3]) (int a, int b)

定义了三个函数指针变量: p[0], p[1], p[2], 它可以指向返回值为 int, 形参为 int, int 的函数

int max(int a, int b) {
    return a > b ? a : b;
}

int main() {
    int (*p) (int, int) = &max;
    
    int a, b;
    a = 1, b = 2;
    
    int d = p(a, b);

    printf("%d\n", d);

    return 0;
}

定义函数指针类型，必须使用typedef，方法就是，在“定义函数指针变量”加上typedef。

typedef int (*p) (int a, int b)

typedef int (*p) (int a, int b);

int max(int a, int b) {
    return a > b ? a : b;
}

int main() {
    p max_2 = max;
    int a, b;
    a = 1, b = 2;
    
    int d = max_2(a, b);

    printf("%d\n", d);

    return 0;
}

一些定义

实例化

实例化（instantiate）是指在的编程中，把用类创建对象的过程称为实例化。是将一个抽象的概念类，具体到该类实物的过程。实例化过程中一般由类名 对象名 = new 类名（参数1，参数2…参数n）构成。
